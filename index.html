HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user
  • Most elements need to have opening and closing tabs
  • It is important to keep on top of these points

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want aroudn the content inside an element.
  • You can make inline CSS code that will effect the element you are working on.
  • It is better to make a separate CSS code file that you can link and make class selectors that will work on that class

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script
  • If you use a function, you must call it in the code in order to execute it
  • Arrays allow us to bunch up a group of data and to log it in a faster way than by doing it one-by-one by using a for loop
  • If statements are useful to see if a variable is what you need or not (true/false)
  • Functions allows us to tell the computer how to do a task, the syntax is available in the notes
